How-to: Add a 3D model

Store your 3D asset

  1. Find the "/data" folder at the root of your project.

You should create the folder if you do not have it.

  1. Choose whether your file should be publicly available or not by choosing the corresponding folder.

You should create the folders if you do not have them.

  1. Add your 3D assets to any subfolder of your choice.

Reference it in the scene

  1. Open your environment. Under your UMI3D Scene node, add an empty object.

  1. Add a UMI3D Model component to that node.

  1. Unwrap the variants parameters and set the size value to 1 to add a variant

  1. Set up the format to the one of the model to link, check the Is a local file checkbox, and browse in your file explorer towards the desired 3D model.

Modify its position, orientation or size

  1. Add an import of the model through Unity to see your model in the editor.

  1. Modify the Transform component on the node that has the UMI3D Model component attached

Troubleshooting

Make sure the node with UMI3D Model component has all its parents up to the node with the UMI3D Scene component having UMI3D Node components.
